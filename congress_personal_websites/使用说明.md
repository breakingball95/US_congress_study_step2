# 美国众议院议员网站爬虫使用说明

## 工具功能
本工具用于爬取美国众议院网站上所有议员的个人官网链接，并将结果保存为Excel文件，方便您分析众议员对中国议题的态度。

### 核心特点
- **多种链接提取策略**：使用三种不同的策略提取众议员个人网站链接
  - 策略1：查找带有 "link is external" 标记的链接
  - 策略2：查找包含 "house.gov" 的链接
  - 策略3：查找所有可能的个人网站链接
- **自动去重**：确保每个众议员只保留一个唯一的网站链接
- **详细的日志记录**：生成 `scraper_log.txt` 文件，记录脚本执行过程和可能的错误
- **错误处理**：包含完整的错误处理机制，确保脚本在遇到问题时能够优雅降级

## 安装步骤

### 1. 安装Python 3
如果您的电脑上还没有安装Python 3，请按照以下步骤安装：

1. 访问 [Python官网](https://www.python.org/downloads/)
2. 下载适合您操作系统的Python 3版本（推荐3.8或更高版本）
3. 运行安装程序，确保勾选"Add Python to PATH"选项
4. 完成安装后，打开命令提示符（CMD），输入 `python --version` 验证安装成功

### 2. 安装依赖包
打开命令提示符（CMD），运行以下命令安装所需的Python库：

```
python -m pip install requests beautifulsoup4 pandas openpyxl
```

### 3. 运行脚本

#### 方法一：使用命令行
1. 打开命令提示符（CMD）
2. 导航到脚本所在目录：
   ```
   cd d:\coding\US_congress_study
   ```
3. 运行脚本：
   ```
   python house_reps_scraper.py
   ```

#### 方法二：双击运行（Windows系统）
1. 找到 `house_reps_scraper.py` 文件
2. 直接双击该文件即可运行

## 输出结果
脚本运行完成后，会在当前目录生成以下文件：

### 1. 主要输出文件
- **house_representatives_websites.xlsx**：包含众议员姓名和个人网站链接的Excel文件
  - **name**：众议员姓名
  - **website**：众议员个人官网链接

### 2. 日志文件
- **scraper_log.txt**：记录脚本执行过程的详细日志，包含：
  - 脚本开始和结束时间
  - 网络请求状态
  - HTML解析情况
  - 链接提取策略的执行结果
  - 数据保存状态
  - 任何可能的错误信息

## 后续使用
1. 打开生成的Excel文件 `house_representatives_websites.xlsx`，查看众议员的个人网站链接
2. 访问感兴趣的议员个人网站，查看其PRESS部分，了解他们对中国议题的态度
3. 根据收集到的信息，分析议员对中国议题的立场

## 常见问题

### 问题1：运行脚本时出现"pip不是内部或外部命令"错误
**解决方案**：
- 确保Python已正确安装并添加到PATH
- 使用 `python -m pip` 代替 `pip`

### 问题2：脚本运行后没有生成Excel文件
**解决方案**：
- 检查 `scraper_log.txt` 文件，查看详细的错误信息
- 确保网络连接正常
- 确保您有足够的权限在当前目录创建文件

### 问题3：Excel文件中没有数据
**解决方案**：
- 查看 `scraper_log.txt` 文件，了解链接提取的情况
- 可能是网站结构发生了变化，导致脚本无法正确提取链接
- 检查网络连接是否正常

### 问题4：脚本运行很慢
**解决方案**：
- 脚本需要从美国众议院网站获取数据，可能会受到网络速度的影响
- 请耐心等待脚本执行完成

## 技术支持

## 文件必要性说明

### 必要文件
以下文件对实现爬虫功能是必需的：
- **house_reps_scraper.py**：主要的爬虫脚本，用于爬取美国众议院议员的个人网站链接
- **使用说明.md**：使用说明文档，提供详细的安装和使用指南
- **运行爬虫.bat**：批处理文件，用于简化脚本的执行过程

### 非必要文件
以下文件对实现爬虫功能不是必需的，可以安全删除：

- `console_test.py` - 测试控制台输出
- `create_file.py` - 测试文件创建
- `simple_test.py` - 简单的Python测试
- `test_env.py` - 测试Python环境
- `test_file_creation.py` - 测试文件创建功能
- `test_file_output.py` - 测试文件输出
- `test_python.py` - 测试Python环境和依赖
- `test_script.py` - 简单的测试脚本

#### 其他爬虫版本
- `simple_scraper.py` - 另一个简化版的爬虫脚本

#### 其他批处理文件
- `run_scraper.bat` - 另一个批处理文件，功能与 `运行爬虫.bat` 重复

### 临时文件
脚本运行过程中会生成以下临时文件，这些文件不是必需的：
- `scraper_log.txt` - 脚本执行的日志文件
- `house_representatives_websites.xlsx` - 脚本生成的Excel文件（这是脚本的输出结果，是否保留取决于您的需求）
